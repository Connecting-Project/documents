# 웹 페이지 계산기 

웹 페이지 계산기는 하와이안 피자에서 제작한 두 번째 프로젝트 입니다. 다음 기능을 지원합니다. 

- 표준 계산기 
- 전역일 계산기 
- 연봉 계산기 
- 경력 계산기 
- 평 계산기 



## Information
